04 — Technical Projects

Things I've
Built & Researched

Featured · Software Engineering
Player Behavior Analysis & Suspicion Scoring Tool
During my internship at Manticore Games, I built an end-to-end player behavior analysis system for Out of Time, a multiplayer roguelike available on the Epic Games Store. The project involved ingesting large volumes of gameplay event data into Google BigQuery, engineering statistical features to characterize player behavior, and designing a scoring algorithm that assigned each account a "suspicion score" based on anomaly detection. I wrapped everything in a Streamlit dashboard so the game dev team could investigate flagged accounts, drill into metric breakdowns, and take action — all without writing SQL. I also delivered stretch goals including explainability metadata (so developers could see why an account was flagged) and an automated data ingestion pipeline that kept the dashboard fresh daily.
